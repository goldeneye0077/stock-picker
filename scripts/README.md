# Scripts Index

## data/
数据采集、补数、生成类脚本。

## diagnostics/
诊断、核对、分析、回归验证脚本。

## dev/
本地开发环境启动/停止/重启脚本。

## migrate/
数据库修复、编码修复、结构迁移相关脚本。

## ops/
运维脚本与仓库结构治理脚本。

## Usage

```bash
python scripts/data/<script>.py
python scripts/diagnostics/<script>.py
node scripts/dev/start-services.js
node scripts/ops/check-root-structure.js
```
